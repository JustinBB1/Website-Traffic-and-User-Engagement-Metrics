Objectif du Projet

Ce projet a pour but de prédire le taux de conversion d'un site web en fonction de ses métriques de performance, utilisant des techniques de Machine Learning et un framework web Django.

Dataset Utilisé

  • Nom : Website Traffic and User Engagement Metrics
  • Source : Kaggle (Anthony Therrien)
  • Enregistrements : 2,000
  • Features : 7 métriques de performance web

Technologies Utilisées

Python
Django
Scikit-learn
Bootstrap 5
